# 会議運営システム (Conference Management System)

Google スプレッドシートと Google Apps Script (GAS) を活用した、2Dコード（二次元コード）ベースの出欠管理システムです。参加者へのコード発行から、当日のスタッフによる読み取り・記録までを一気通貫で行えます。

## 📋 主な機能

### 1. 参加者向け機能
- **二次元コード発行**: 団体を選択して、自分専用の参加用コードを発行。
- **メール通知**: 発行と同時に、登録されたメールアドレスへコードを自動送付。
- **重複発行防止**: 1つの端末（ブラウザ）からは1つの団体分しか発行できない制限。

### 2. スタッフ向け機能
- **秘密の管理画面**: URLパラメータ（`?sample=value`）によるアクセス制限。
- **受付モード管理**: 会議マスタから当日対象の会議を選択して受付を開始/停止。
- **2つの受付方法**:
  1. **標準カメラ**: コードを読み取ってURLを開くだけで受付完了。
  2. **写真解析**: 画面上で撮影した写真やスクリーンショットを解析して受付（`jsQR`ライブラリ使用）。
- **再発行対応**: 紛失や誤発行があった際、スタッフ画面から強制的にコードを更新。

### 3. データ連携
- **マスター同期**: 別ファイル（団体名簿・マスターファイル）との `IMPORTRANGE` 関数による自動データ連携。

## ⚙️ セットアップ手順

### 1. スプレッドシートの準備
以下のシートを作成してください。

#### シート名: `名簿`
| A: ID | B: 団体名 | C: 責任者 | D: メールアドレス | E: トークン | F: 発行日時 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| (任意) | (表示名) | (メール宛名) | (送信先) | (システム自動入力) | (システム自動入力) |

#### シート名: `出欠記録`
- ヘッダーのみ作成（打刻時に自動追記されます）。

#### シート名: `会議マスタ`
- A列に開催予定の会議名をリストアップしてください。

### 2. Google Apps Script のデプロイ
1. スプレッドシートの「拡張機能」>「Apps Script」を開きます。
2. `コード.gs` と `index.html` を作成し、ソースコードを貼り付けます。
3. 右上の「デプロイ」>「新しいデプロイ」を選択。
   - **種類**: ウェブアプリ
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: 全員
4. 発行された **ウェブアプリのURL** を控えてください。

## 🚀 運用方法

### スタッフの準備（各会議ごと）
1. スタッフ用URL（`アプリURL?sample=value`）にアクセスします。
2. 「会議名」を選択し、「この会議の受付を開始する」をクリックします。
   - これにより、サーバー側で受付フラグが立ち、スタッフの端末が「受付専用機」として認証されます。

### 参加者の操作
1. 標準URLにアクセスし、自身の団体を選択して発行します。
2. 表示されたコードを保存するか、メールを確認します。

### 当日の受付
- スタッフが自身のスマホのカメラ、または管理画面の「写真から受付」機能を使って参加者のコードをスキャンします。
- **注意**: 参加者が自分のスマホでURLを開いても、スタッフ認証がないため受付は完了しません。

## 🔒 セキュリティとプライバシー

- **個人情報保護**: 参加者用プルダウンには「団体名」のみを表示し、責任者名はスタッフ画面でのみ表示されます。
- **トークン方式**: 2Dコードには団体IDではなく、推測不可能なUUID（トークン）が含まれています。
- **サーバーサイド認証**: 現在どの会議が受付中かはサーバー（PropertiesService）で管理しており、受付停止後は一切の打刻が受け付けられません。
- **端末固定**: `localStorage` を使用し、一般参加者が他団体のコードを次々と発行することを抑制しています。

## 🛠 使用技術
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Google Apps Script (GAS)
- **Library**: [jsQR](https://github.com/cozmo/jsQR) (QRコード解析)
- **API**: [QR Server API](https://goqr.me/api/) (QRコード生成)
